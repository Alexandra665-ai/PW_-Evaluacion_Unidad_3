# Adopta un Amigo

Mini aplicación web desarrollada para la asignatura **Programación Web**.

Su objetivo es permitir publicar mascotas en adopción, visualizar las mascotas disponibles y facilitar el contacto entre los interesados y el responsable de cada publicación.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Node.js
- LocalStorage

---

## Funcionalidades

- Publicar una mascota en adopción.
- Validación de todos los campos del formulario.
- Subir una fotografía de la mascota.
- Buscar mascotas por nombre.
- Filtrar por tipo de animal.
- Filtrar por estado de adopción.
- Mostrar estadísticas:
  - Total de mascotas.
  - Mascotas disponibles.
  - Mascotas adoptadas.
- Guardar la información utilizando LocalStorage.
- Solicitar la adopción mediante un formulario de contacto.

## Instalación

1. Clonar el repositorio.

```
https://github.com/Alexandra665-ai/PW_-Evaluacion_Unidad_3
```

2. Entrar a la carpeta del proyecto.

```
cd Evaluacion_Unidad_3
```

3. Instalar las dependencias.

```
npm install
```

4. Ejecutar el servidor.

```
node server.js
```

5. Abrir el navegador.

```
http://localhost:3000
```
